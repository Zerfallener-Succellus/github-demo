Teste Git Quick Start demo

git init "" = inicia um repositorio novo

git add "."(todos os arquivos) // "nome do arquivo" = coloca o arquivo na "fila de commit"

git status = retorna os arquivos mostrando quais foram ou não commitados

git commit = commita as mudanças, caso deixado em branco abrira um VIM para que sejá escrita a mensagem de commit (DICA: esc,esc sai do modo de edição, :w salva e :q sai, :wq salva e sai)

git push = coloca os arquivos commitados no repositorio
